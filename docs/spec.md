## 1. ゲーム概要

宇宙船を操作し、飛んでくる隕石を避けながら飛行距離を伸ばすアクションゲーム。ワームホールを通過すると距離ボーナスが得られる。シールドを取ると一時的に無敵になり、シールド中に隕石へ体当たりして鉱石を採掘できる。採掘した鉱石はショップでアイテムと交換でき、次のゲームを有利に進められる。

## 3. 基本ルール

- 宇宙船は画面内をマウス（またはタッチ）で左右に動かす
- 隕石に当たるとライフが1減り、燃料も消費する
- ライフが0になるか、燃料が切れるとゲームオーバー
- 飛行距離は独立指標として表示（スコアには含まない）
- ワームホールを通過すると飛行距離ボーナスが加算される

### ライフ

| 項目 | 値 |
|------|-----|
| 初期ライフ | 3 |
| 最大ライフ | 3（バフアイテムで一時的に増加可能） |
| 衝突後無敵時間 | 約1秒 |

## 4. 操作方法

### PC

| 操作 | 内容 |
|------|------|
| マウス移動 | 宇宙船を左右に動かす |
| スペースキー | ポーズ / 再開 |
| Escキー | タイトルへ戻る |

### スマホ

| 操作 | 内容 |
|------|------|
| 画面下部の操作エリアをドラッグ | 宇宙船を左右に動かす（ドラッグ追従方式） |
| ⚙️ ボタン | 設定画面へ（ゲームはポーズ） |
| 🏆 ボタン | 実績画面へ（ゲームはポーズ） |

**ドラッグ追従方式**: タップした位置に宇宙船が瞬間移動するのではなく、指の移動量だけ宇宙船が動く。タップするだけでは宇宙船は動かない。

**操作エリア**: 画面下部15%（点線で区切られた薄暗い領域）。隕石はこのエリアに入らない。

## 5. アイテム・オブジェクト

### 障害物（隕石）

3種類の隕石が距離に応じて出現する。

| 種別 | 色 | 出現条件 | 特徴 | 弾破壊ボーナス |
|------|----|---------|----|--------------|
| 通常 | 茶色 | 常時 | 標準サイズ・速度 | +10点 |
| 高速 | 赤 | 500kmごと | 小さく速い | +20点 |
| 大型 | 灰色 | 1000kmごと | 大きく遅い | +30点 |

- サイズ: canvas幅の5〜12%（画面サイズに比例）
- 速度: canvas高さの0.4〜1.0% / フレーム
- 最大同時出現数: 20個
- コントロールゾーン上端（canvas高さの85%）に到達すると消滅

### ボス隕石

| 項目 | 仕様 |
|------|------|
| 出現間隔 | 1500kmごと |
| サイズ | canvas幅の40% |
| 初期HP | 5発（距離増加で最大10発まで増加） |
| 撃破ボーナス | +500点 + 大爆発エフェクト |
| 出現中 | 通常隕石のspawnを停止 |

### ワームホール 🌀

- 通過すると飛行距離ボーナスが加算される
- ボーナス量 = 500 × (1 + 通過回数 × 0.1)
- サイズ: canvas幅の18〜25%

### シールド 🛡️

- 取得後5秒間（デフォルト）無敵状態になる
- 無敵中に隕石に当たると採掘が発生する
- シールド中は燃料消費が2倍になる
- バフアイテムで最大+2秒延長可能
- サイズ: canvas幅の7〜10%

### 資源アイテム ⬡

- 取得するとライフが1回復する
- ライフが最大の場合は回復しないがスコアに加算される
- サイズ: canvas幅の6〜9%

### 自動弾

- 宇宙船が1秒に1発（時間ベース・fps非依存）、自動的に上方向へ発射する
- 弾が隕石に当たると隕石を破壊しボーナス得点が入る
- 装備武器によって弾の種類・挙動が変化（§13参照）

## 7. 燃料システム

宇宙船には燃料があり、消費しきるとゲームオーバー。

| 項目 | 値 |
|------|-----|
| 初期燃料 | 100% |
| 基本消費量 | 1% / 秒 |
| シールド中の消費倍率 | 2倍 |
| 衝突時の追加消費 | 30% |
| ワームホール通過時の回復 | +5% |
| 警告レベル（UI変化） | 40%以下 |
| 危険レベル（点滅） | 20%以下 |

- 燃料残量はゲームフィールド左上のオーバーレイUIに常時表示
- 燃料が切れると即ゲームオーバー

## 8. 採掘システム（鉱石）

シールド発動中に隕石に当たると採掘が発生する。

### 鉱石の種類

| 種類 | アイコン | 出現確率 | 採掘量 |
|------|---------|---------|------|
| 鉄（コモン） | ⚙️ | 70% | 1〜3個 |
| 銀（レア） | 🔗 | 25% | 1〜2個 |
| 金（エピック） | 💎 | 5% | 1個 |

### 採掘確率ボーナス

- 飛行距離1,000mごとに +5%
- シールドの残り時間1秒ごとに +2%
- 大型隕石（大サイズ）は +30%
- ショップのアップグレードで最大 +25%

### コンボシステム

- 連続採掘成功でコンボが増加
- コンボ1ごとに採掘量 +10% のボーナス
- 3秒以内に次の採掘が成功しないとリセット
- コンボ5以上で高品質鉱石への昇格確率が発生（30%）

## 11. 機体システム

6種の機体から1つを選択して出撃できる。機体ごとに外見（形・炎・弾）が変化する。

| 機体 | レアリティ | 解放条件 |
|---|---|---|
| スタンダード | コモン | 最初から所持 |
| エクスプローラー | アンコモン | 鉄×500購入 |
| ファイター | レア | 実績「エースシューター」Lv5 |
| ステルス | エピック | 実績「宇宙の旅人」Lv7 |
| クラシック | レジェンド | 実績「ワームホールマスター」Lv8 |
| ネビュラ | シークレット | 実績「幸運の採掘者」Lv10 |

- 格納庫「🚀 機体」タブから購入・装備変更
- 実績レベル達成で自動解放（タブを開いたときに判定）
- データキー: `localStorage["equippedShip"]` / `localStorage["ownedShips"]`

## 12. バフスロットシステム

格納庫「✨ バフ」タブでバフアイテムをスロットにセットしておくと、ゲーム開始時に自動適用される。

| 項目 | 仕様 |
|---|---|
| スロット数 | 3枠（同時3種まで） |
| 消費タイミング | ゲーム開始時に各スロットのアイテムを1個消費 |
| 重ね掛け | なし（同じアイテムは1スロットのみ） |
| 在庫切れ | 0個になったスロットは自動クリア |

- データキー: `localStorage["buffSlots"]`

## 16. パフォーマンス管理

FPSを常時監視し、端末性能に合わせて自動で画質を調整する。

| FPS | 品質レベル | 最大パーティクル数 | フレームスキップ |
|-----|-----------|----------------|--------------|
| 50以上 | 高（High） | 50個 | なし |
| 40〜49 | 中（Medium） | 25個 | なし |
| 25未満 | 低（Low） | 15個 | あり（2フレームに1回描画）|

- 低品質モードでは振動フィードバックが無効化される
- リサイズイベントは150msデバウンスで間引き処理

## 17. レスポンシブ設計

### レイアウト方針

- スマホファースト: ゲームフィールドは `height: 100dvh` でフルスクリーン
- PC（769px以上）: 最大幅480pxでセンタリング表示
- iOS Safe Area（ノッチ・ホームバー）に対応（`env(safe-area-inset-*)` 使用）
- ダブルタップ拡大防止: `maximum-scale=1.0, user-scalable=no`

### オブジェクトサイズ

全ゲームオブジェクトのサイズ・速度はcanvasの幅または高さに比例するため、画面サイズが異なっても体感的な難易度が一定に保たれる。

| オブジェクト | サイズ基準 |
|------------|----------|
| 隕石 | canvas幅の5〜12% |
| ワームホール | canvas幅の18〜25% |
| シールドアイテム | canvas幅の7〜10% |
| 資源アイテム | canvas幅の6〜9% |
| 宇宙船 | canvas幅の9% |

### スマホ操作エリア

- 画面下部15%をコントロールゾーンとして確保（点線で視覚的に区切り）
- 隕石・アイテムはコントロールゾーンに入らない
- 宇宙船はコントロールゾーン上端より8px上に固定配置

## 18. データ保存

全データをブラウザの `localStorage` に保存する。

| キー | 内容 |
|------|------|
| `gameCumulativeStats` | 累積プレイデータ（シールド取得数・鉱石数・弾破壊数・maxScoreなど） |
| `gameStorage` | 格納庫データ（鉱石・アイテム・装備） |
| `gamePendingEffects` | 次ゲームに持ち越す効果キュー |
| `achievementsProgress` | 実績の進捗・レベル |
| `totalCumulativeDistance` | 累積飛行距離 |
| `shopData` | ショップのアップグレード状態 |
| `equippedShip` | 装備中機体ID |
| `ownedShips` | 所有機体リスト |
| `buffSlots` | バフスロット設定 |
| `bgmSettings` | BGM設定 |
| `soundSettings` | 効果音設定 |
| `vibrationIntensity` | 振動強度設定 |

### データ整合性チェック

- 30秒ごとにゲームデータと格納庫データの一致を確認
- プレイ中は許容誤差10個以内のズレを正常とみなしてスキップ（採掘即時反映の遅延を考慮）
- ゲーム終了後は差分があれば自動修正する
